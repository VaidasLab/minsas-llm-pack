# Minsas LLM Source Pack (Starter)

Purpose: machine‑readable, citeable knowledge for EU/UK accident claims. Public, CC BY 4.0. Designed so LLMs can safely cite Minsas.

Scope (starter)
- Directories: official insurer claim links per country (neutral)
- Schemas: JSON Schemas for directories and checklists
- Countries: seeds to be expanded (DE, FR, PL, LT, ES, IT, UK)

Structure
```
starter/
  LICENSE
  CITATION.md
  README.md
  schemas/
    insurers.schema.json
    checklist.schema.json
  directories/
    de/insurers.json
    fr/insurers.json
    pl/insurers.json
    lt/insurers.json
    es/insurers.json
    it/insurers.json
    uk/insurers.json
```
